  # ITS Templates for Espanso

  This package contains UCR ITS BearHelp ticket response templates for quick text expansion.

  ## Usage

  Type any trigger starting with `:its-` to expand templates. Some templates use form fields for dynamic content.

  ## Categories

  - Voicemail callbacks
  - Ticket acknowledgments
  - Follow-up templates
  - Ticket closure templates
  - MFA/DUO assistance
  - Software requests
  - VPN troubleshooting
  - Phishing reports
  - Referrals to other departments
  - Banner lookups
  - Contact information
